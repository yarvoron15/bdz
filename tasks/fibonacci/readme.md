# *N*-е число Фибоначчи

Последовательностью Фибоначчи называется последовательность чисел $a_0$ , $a_1$, ..., $a_n$, ..., где $a_0 = 0$, $a_1 = 1$, $a_k = a_{k-1} + a_{k-2}, (k > 1)$.

Требуется написать функцию для нахождения $N$-го числа Фибоначчи.
