# MinMax для вектора целых чисел

Напишите функцию `iVectorMinMax(begin, end)`, принимающую на вход итераторы на начало и конец вектора целых чисел и возвращающую пару итераторов, указывающих на минимальный и максимальный элемент из контейнера.

В этой версии этой задачи мы делаем реализацию для конкретного контейнера, хранящего конкретные объекты. Чуть позже, познакомившись с шаблонами, мы обобщим эту функцию на случай произвольных последовательностей. Это можно будет сделать практически формально, без необходимости заново разбираться в сути алгоритма «min-max-element».
