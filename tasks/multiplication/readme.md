# Умножение чисел

Эта задача предназначена для знакомства с тестирующей системой. Решение этой задачи не учитываются в итоговой оценке.

Вам предстоит реализовать функцию `multiply` в файле `multiplication.cpp`. Функция перемножает два целых числа.. 

### Структура задачи

В каждой задаче обычно есть как минимум 3-4 файла:

* `CMakeLists.txt` — CMake-файл для сборки задачи;
* `test.cpp` — модульные тесты для проверки корректности решения;
* `*.h` — некоторый заголовочный файл, который содержит объявления функций/классов, которые необходимо реализовать;
* `*.cpp` — модули трансляции, в которых вы должны реализовать то, что требуется по условию задачи (в данном примере — `multiplication.cpp`).

Вам разрешено изменять **только** файлы, содержащие решение задачи, — в противном случае проверочная система может не принять вашу посылку.
Когда вы делаете посылку, система запускает `test.cpp` и дополнительные скрытые тесты (опционально). Прохождение тестов является необходимым, но не достаточным условием, чтобы засчитать задачу. Преподаватель/учебный ассистент может проверить ваш код и аргументированно отказать в зачете решения. В этом случае вы можете исправить решение и загрузить его снова.

### Линтеры

В работе с кодом в индустрии очень хорошо иметь линтер, который по заданным правилам приводит ваш код к единому стилю.
В курсе мы будем использовать [C++ Google Code Style](https://google.github.io/styleguide/cppguide.html) с некоторыми модификациями, например:

* исключения можно (а иногда — и нужно) использовать;
* отступы — 4 пробела;
* максимальная длина строки в 100 символов.

Если ваш код не будет соответствовать кодстайлу, решение может быть не засчитано или оценка за него снижена. Чтобы привести его к стилю, используйте `clang-format -i file.cpp`, конфигурацию `clang-format` можно прочитать [здесь](../../.clang-format).
